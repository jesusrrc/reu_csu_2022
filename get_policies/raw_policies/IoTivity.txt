<p>IoTivity is an open source software project enabling seamless device-to-device connectivity where billions of wired and wireless Internet of Things (IoT) devices can securely connect to each other and to the internet.  The Open Connectivity Foundation (OCF) develops specifications, interoperability guidelines, and a certification program for these devices. IoTivity Classic and IoTivity Lite are open source reference implementations of the OCF specification. You can learn more <a href="https://openconnectivity.org/" rel="noopener" style="background-color: #ffffff;" target="_blank">about OCF here</a>,  and <a href="http://iotivity.org" rel="noopener" style="background-color: #ffffff;" target="_blank">about IoTivity here</a>.</p>
<p>The IoTivity Lite framework APIs expose the framework to developers, and are available in several programming languages and for multiple operating systems. The framework supports dedicated and optimized protocols for IoT, with specific considerations for constrained devices and addressing all types of devices, form-factors, companies, ecosystems and markets.</p>
<p>The IoTivity Lite framework operates as middleware across all operating systems and connectivity platforms and has these four essential building blocks:</p>
<ul>
<li><strong>Discovery: </strong>supporting multiple mechanisms for discovering devices and resources in proximity and remotely.</li>
<li><strong>Data transmission: </strong> supporting information exchange and control based on a messaging and streaming model.</li>
<li><strong>Data management: </strong> supporting the collection, storage and analysis of data from various resources.</li>
<li><strong>Device management: </strong> supporting configuration, provisioning and diagnostics of devices.</li>
</ul>
<p>See below for information on how to get started with the <a href="https://iotivity.org/getting-started" rel="noopener" target="_blank">IoTivity Lite</a> code base.</p>
