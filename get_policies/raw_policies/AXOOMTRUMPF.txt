<div class="gr-12">
<h2 class="headline-small">
                                                    Network Conditions
                                                </h2>
<div class="ux-text-normal ux-u-margin-children-0">
<p>To be able to use the "TRUMPF Machine Apps" machine option, TRUMPF machines are made connectable to the AXOOM cloud platform. To do this, the customer network has to fulfill the following network requirements:</p><ul class="list-styled"> <li>Connection of the machine(s) to the Internet via the network. As a minimumrequirement, the following URLs must be reachable: 	<ul class="list-styled"> <li><a href="https://trumpf.axoom-gate.com" target="_blank">https://trumpf.axoom-gate.com</a> and</li> <li><a href="https://trumpf-provisioning.myaxoom.eu" target="_blank">https://trumpf-provisioning.myaxoom.eu</a> </li> </ul> </li> <li>Enabling of TCP/443 (Port 443) (proxy and/or firewall) for the machine(s)</li> <li>Enabling of WSS network protocol (access from LAN to the Internet)</li> <li>Enabling of HTTPS network protocol (access from LAN to the Internet)</li> <li>With an Internet connection over a proxy server, the following information must be available for the connection: The host name of the proxy server, IP address of the proxy server, user name of the proxy server for the machine(s), password of the proxy server for the machine(s), port for the proxy server.</li> </ul>
</div>
</div>
</div>
</div>
</div>

<div class="accordeon-entry-panel js-accordeon-toggle">

<div class="gr-12">
<div class="ux-text-normal ux-u-margin-children-0">
<p>The following interfaces are necessary to enable the connection to the Condition Monitoring Platform:</p><ul class="list-styled"> <li>LAN port</li> <li>Ethernet IEEE 802.3 10/100-BaseTX, RJ 45, Full Duplex, Auto-MDIX</li> </ul><ul class="list-styled"> <li><strong>Retrofit Cloud Connector</strong> <ul class="list-styled"> <li>Enabling of TCP/443 (Port 443 and 8443) (proxy and/or firewall) for the machine(s)</li> <li>If the machines are connected via the Retrofit Cloud Connector, the following URLs must be reachable: <a href="https://trumpf-us-emsg.cloud.thingworx.com" target="_blank">https://trumpf-us-emsg.cloud.thingworx.com</a> </li> </ul> </li> </ul><ul class="list-styled"> <li><strong>Logfile Collector</strong><br/> 	If the machines are connected via the Internet, the LAN needs to enable UDP port 500 and UDP 4500 for access from the Internet. For possible questions regarding the customer network, a contact person for IT / network administration is needed.</li> </ul>
